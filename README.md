# BioLogist Backend (Django)

## 📌 Description
Backend for an e-commerce platform built using Django and Django REST Framework.

## ⚙️ Features
- REST API for product management
- Cart and order handling
- User authentication

## 🛠️ Tech Stack
- Django
- Django REST Framework
- MySQL / SQLite

## 🗄️ Models
- User
- Product
- Cart
- Order

## 🔗 API Endpoints
- GET /products/
- POST /cart/
- POST /order/

## ▶️ Run Project
pip install -r requirements.txt
python manage.py runserver
